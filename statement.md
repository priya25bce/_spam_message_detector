Problem Statement : -

Technology is growing rapidly so now we have shifted everything to online. We have started communicating through email, SMS and messaging apps. With great features comes great risks. The issuse of spam messages is increasing rapidly. We often cannot decide whether the message is or not. And if we click on spam messages it can have serious problems such as security and personal information being shared with them.

To solve this problem, I have made the project on "Spam Message Detector." This project helps the user to identify the message whether the message is spam or not. You can type the message in the terminal and get the answer there (CLI Interface) or you can also check it on the GUI Window (GUI Interface) according to your wish.

Scope of the project :-

Tells the message is sapm or not using maching learning techniques.

It preprocesses the text

It trains the model on the given spam dataset

Target Users :-

Students who are learning machine learning.

People who are using mobile phones

People using emails

Developers

Educators

High- Level Features :-

It tells you whether a message is spam or not.

It uses TF-IDF for preprocessing of data and Naive Bayes Algorithm which helped to classify.

It has both CLI interface and GUI Interface.

It is fast and easy to use.
